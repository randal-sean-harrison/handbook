Database

When adding a database, most information will be provided for you. We are working to standardize and synchronize database descriptions across Library platforms. However, you may use your own discretion in customizing the database description where appropriate.

Requirements

Book

2 Edit the Description, if necessary. Optional
3 Select the Display beneath item title radio button
