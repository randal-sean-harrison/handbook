DOs and DON'Ts

These are some of the most important guidelines for ensuring a consistent voice and brand identity for the guides, as well as a consistent experience for the user.

Don't publish blank pages This is a common accidental effect when creating sub-pages. The main landing page, if blank, should either:
  • Redirect to the first sub-page. Recommended
  • Offer a table of contents with links to each subpage.
Don't change font family or size Except in certain cases, such as foreign languages or special text formatting, do not change the font family or font size for your guides. See the typography guidelines for more information.
Don't use <H1> or <H2> headings For accessibility reasons, don't use these heading levels in your rich text boxes.
Don't paste text directly into the editor Use the Pasrte from Word function
