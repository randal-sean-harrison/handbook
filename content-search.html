Google Search

The Google Search box is a simple search box serving 4 different Google services: Google Scholar, Google Books, Google Patent, and the standard Google Web search. The only customization option is to prepopulate the search field with one or more terms.

Request a Search Box
Additional search engines are also available to be mapped into your guide. Find them in the Reusable Assets Guide. If you would like to request that a customized search box, please contact us.

Requirements

Follow these steps to ensure the minimum requirements when adding your Google search box.

Google Search Box
1 Select the Google service you wish to search from the Search Type dropdown.
2 Add one or more terms to the Default Query field for a default search. Optional
3 Check the Yes option if presented with the option to Use Proxy.
4 Select the appropriate order among other box content from the Position dropdown.
