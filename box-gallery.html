Gallery Box

A gallery box is a way to present images and other library resources in an interactive linear progression. In addition to images, you may add books, databases, library guides, and LibCal events.

You may customize your gallery in the following ways:

  • A transparent or colored caption, with rich text paragraph and hyperlinked title, at top, bottom, left or right.
  • A simple navigation menu at top, bottom, left or right.
  • Transition type may be slide or fade.
  • Animation options include autoplay and a specific transition speed.
  • An image gallery can be made by setting multiple images per row, across multiple rows.

Example, "Images"

Standardize Your Size
Make sure that the images you use in your slides all have at least the same height, if not both the width and height.

Example, "Instructions"

This instruction gallery requires the user to advance to the next slide. Each slide represents a step in a linear process. As the name of the step, the title of the slide is not linked. The description field is used to describe the step further.
