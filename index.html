Guide Requirements

Before You Create that Guide...

General Subject Guides curate Library and other research-related resources oriented around prominent disciplines at Notre Dame.

Contact the LMT first...
General Subject guides are managed by the LibGuides Management Team. Please fill out the Guide Proposal Form before creating a General Subject guide.

Before You Create that Guide...

Specific Subject Guides curate Library and other research-related resources, narrowing and further focusing the topic of General Subject Guides.

Before You Create that Guide...

A Course Guide curates Library and other research-related resources for a specific Notre Dame course.

Guide Proliferation Policy
Only the most recent version of any instructor's particular Course Guide may be published. Versions from previous semesters/years must be unpublished.

Before You Create that Guide...

How-to Guides instruct users in using a particular resource(s) or perfoming a specific process.

Contact the LMT first...
How-to guides are managed by the LibGuides Management Team. Please fill out the Guide Proposal Form before creating a How-to guide.

Before You Create that Guide...

Reference Guides curate resources useful across multiple academic disciplines. General Purpose Guides present skills/topics that are general in nature.

Contact the LMT first...
Reference / General Purpose guides are managed by the LibGuides Management Team. Please fill out the Guide Proposal Form before creating a Reference / General Purpose Guide.

Before You Create that Guide...

Internal Guides are private guides which may adopt any organization scheme or style.

All Internal Guides must be set to Private. Passwords are required only for guides containing sensitive information.

Box Title Format
Box titles must be title case. They should not contain labels or special icons.
